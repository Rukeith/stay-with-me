## Env

* `FLASK_APP`：app.py `不能為 flask.py`，指定 entry file
* `FLASK_ENV`：`development` 將會開啟 reload、debug mode 和 debugger
* `FLASK_DEBUG`：`1` 控制 debug mode
* `FLASK_RUN_PORT`：flask 的 port
